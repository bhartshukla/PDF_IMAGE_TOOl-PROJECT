# 🚀 XBzin File Tools

A modern, fast, and secure **all-in-one file converter web app** built using pure HTML, CSS, and JavaScript.

Convert images and PDFs directly in your browser — **no uploads, no server, 100% private**.

---

## 🌐 Live Features

* 🖼️ **Image Converter**

  * Convert between JPG, PNG, WebP, BMP
  * Adjustable quality settings
  * Batch conversion support

* 📄 **PDF → Images**

  * Extract each page as PNG/JPG
  * High-quality rendering using PDF.js
  * Real-time progress tracking

* 🖼️➡️📄 **Images → PDF**

  * Merge multiple images into a single PDF
  * Drag & drop reordering
  * A4 / Letter / Fit-to-image support

* 📦 **Bulk Download**

  * Download all files as ZIP (JSZip + FileSaver)

---

## 🔒 Privacy First

> ✅ All processing happens locally in your browser
> ❌ No file uploads
> ❌ No server interaction

Your files never leave your device.

---

## ⚡ Tech Stack

* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Libraries:**

  * PDF.js (PDF rendering)
  * JSZip (ZIP download)
  * FileSaver.js (file saving)
  * jsPDF (PDF creation)
* **Icons:** Remix Icons
